# Requirements

#### As a User I want to...

1. Be able to get an overview over all books registered
2. Be able to get an overview over all books registered in a specific box
3. Scan a QR code on the box to access that boxes interface
4. Have a map that shows me the locations of the boxes and be able to click on a box to access its catalogue of registered books
5. Be able to scan a book's EAN code to add a book to the box or to remove a book from the box
6. See when a book has been last scanned (colouring)
7. Be able to scan several books to update the catalogue (game)
8. Try to find books that weren't scanned in a long time (game)
9. Be able to help the library by scanning a few books to maintain the up-to-dateness of the catalogue.
10. Be able to get a quick overview over a certain book by scanning it.
